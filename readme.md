
## Simple example of Testing Framework using Java, Maven, TestNg.
#### Clone project:
```
git clone https://github.com/vsshk/java-maven-testng-example-simple.git
cd java-maven-testng-example-simple
```
### How to run
```
$ mvn -Dgroups=Smoke,Regression test
```
or
```
$ mvn test 
```
